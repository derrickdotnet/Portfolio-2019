React.js Landing Page UI


I designed a simple landing page for an app to be developed using react.js.

I focused on a mobile first design. I designed it to where there are react components can be used repeatedly the app: The header and the footer components can be used throughout. Eachpage can have its own body and with components and fragments o be used in the app.

I first made sketches with pencil and paper to create a starting point for my ideas. In my sketches I wrote down possible functions of components and played around with some thoughts. From there I created wireframes, then made high-res mockup in Adobe XD.
